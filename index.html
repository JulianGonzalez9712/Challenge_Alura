Encriptador de Texto - Proyecto ONE Alura Latam y Oracle

Este encriptador de texto fue desarrollado como parte del programa ONE de Alura Latam y Oracle por parte de Julian Andres Gonzalez. Su objetivo es proporcionar una herramienta sencilla para la codificación y decodificación de mensajes, aplicando conceptos clave en lógica de programación y criptografía básica.
